# Security Policy

## Supported Versions

The latest `main` branch and the latest stable release tag are supported for security fixes.

## Reporting a Vulnerability

If you find a security issue in Auto Pilot, please do not open a public issue first.

- Email: `choi.ms.dev@gmail.com`
- GitHub: [open a private security advisory](https://github.com/minsu0707/auto-pilot/security/advisories/new)

Please include:

- a short description of the issue
- affected files or flows
- reproduction steps
- potential impact

I will review reports as quickly as possible and coordinate a fix before public disclosure.
