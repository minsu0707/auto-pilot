# Privacy Policy

Auto Pilot is distributed as a local Codex plugin.

## Data handling

- Auto Pilot stores project intake answers and runtime state in the local workspace you run it against.
- Auto Pilot does not include its own hosted backend or remote telemetry service.
- Installation downloads plugin files from the published GitHub repository.

## User responsibility

- Review any prompts, secrets, API keys, or production credentials before using them with Codex.
- Do not store sensitive data in shared repositories unless that is intentional.
